# README　このアプリについて
***

## :grinning:STORKの模写とは
WordPressのテーマであるSTORKの模写コーディング(個人制作)
<br></br>
メディアクエリを使った、レスポンシブ対応。jQueryを使用しハンバーガーメニューや複数表示スライドショーやドロップダウンメニューやページ内遷移機能や検索モーダルなどを実装。
